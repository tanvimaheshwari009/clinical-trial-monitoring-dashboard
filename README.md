# Clinical Trial Monitoring Dashboard (Power BI)

## Overview
This project showcases a Power BI dashboard designed to monitor key operational metrics in clinical trials.

## Key Metrics Tracked
- Visit Compliance %
- Total Delayed Visits
- Protocol Deviation Flag
- Site-level compliance comparison
- Latest active site, enrolled patient, and visit

## Tools & Skills
- Power BI
- DAX
- Clinical Trial Operations Metrics
- Data Visualization & Monitoring
